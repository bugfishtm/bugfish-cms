📁 **Administrator Releases Folder**

In this folder you will find releases of the _administrator module. The latest version of this module is always implemented in the latest bugfishCMS Release but can be obtained and updated manually. 

So you can redeploy the administrator module multiple times or copy it and use the module for your own purposes. You can also update it manually if necessary or fetch old versions if you need them.
